## Descripción de la Actividad

El objetivo de esta actividad es familiarizarse con el uso de herramientas básicas de desarrollo y despliegue utilizadas en proyectos de análisis de datos e inteligencia de negocios. En particular, se trabajó con:

- **GitHub** para el control de versiones y administración del proyecto.
- **Streamlit** para la creación y despliegue de aplicaciones web interactivas en Python.

Esta práctica permite comprender cómo organizar proyectos, compartir código y publicar aplicaciones de manera sencilla.
